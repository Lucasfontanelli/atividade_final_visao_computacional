# Sistema de Inspeção Visual de Laranjas

Projeto de visão computacional para classificar laranjas saudáveis e podres usando segmentação, extração de features manuais e classificadores clássicos.

## Cenário escolhido

Cenário A - Inspeção de frutas.

## Classes

- orange: laranjas saudáveis
- rottenoranges: laranjas podres
